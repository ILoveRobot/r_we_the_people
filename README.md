R We The People?
----------------

An R Package for working with We The People petition data.

The package is currently focused on loading petition data,
running ad-hoc analysis and exploratory visualizations.

Usage
-----

    install.package('wethepeople')

    library(wethepeople)
    example(wethepeople)
